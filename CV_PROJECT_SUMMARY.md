## Legal RAG Chatbot

• Built AI-powered legal research assistant querying 1,671+ court cases using Claude Sonnet 4.5, OpenAI embeddings, and Pinecone vector database with RAG architecture
• Developed React/TypeScript frontend and FastAPI backend with real-time chat interface, case viewer, and PDF generation capabilities
• Deployed containerized microservices to Azure Kubernetes Service (AKS) with automated CI/CD pipelines using Azure DevOps, Docker, and Azure Container Registry
